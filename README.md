## Deskripsi
Proyek Klasifikasi Gambar ini mendemonstrasikan implementasi teknik-teknik lanjutan dalam pengembangan model Machine Learning (ML), khususnya pada domain **Computer Vision**. Fokus utama proyek ini adalah membangun model klasifikasi gambar yang akurat dan siap dideploy di berbagai platform.

Saya sendiri  disini memilih untuk membuat model Fish Clasification dengan 36 jenis yang mampu di clasifikasikan oleh model ini pada akhir nya yang menggunakan CNN untuk pelatihan nya.
